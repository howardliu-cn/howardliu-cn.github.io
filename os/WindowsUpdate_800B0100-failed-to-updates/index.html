Windows 7 系统更新失败-WindowsUpdate_800B0100

2014-12-02
阅读量 0

这几天更新IE，总是有一个WindowsUpdate_800B0100的失败，废了半天劲才解决，记录一下。

1 问题描述

  这几天把IE更新到了IE11，然后系统自动检测到一个重要更新：用于基于 x64系统的 Windows 7 的 Internet Explorer 11 的累积安全更新程序(KB3003057)。

  既然检测到，而且是安全更新，那就直接安装（虽然我平时用Chrome）。结果，尝试多次，都报更新失败，错误代码8024000B。联机获取帮助后，得到的更新提示为：”WindowsUpdate_800B0100” “WindowsUpdate_dt000” failed to install updates。

2 解决方法

折腾了半天，终于找到解决办法：

  1. 进入C:\Windows\SoftwareDistribution\Download文件夹
  2. 删除所有文件

哦了，可以重新安装更新了。

3 问题深入

  对于Windows Update 错误 800B0100，说明是在更新过程中，所需的某个文件已破坏或缺失。回想一下，确实自己在更新阶段有过强制关机，估计就是在那个时候使文件损坏了（no zuo no die）。

  微软官网对于这个问题有详细描述：Windows Update 错误 800B0100，然后还提供了一个系统更新就绪工具用于解决更新问题。
